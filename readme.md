# Spendly (React-Native and Expo v54.0.0)

Open the `App.js` file to start writing some code. You can preview the changes directly on your phone or tablet on **Expo go app** by scanning the **QR code** or use the iOS or Android emulators. When you're done, click **Save** and share the link!
