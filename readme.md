# Spendly (React-Native and Expo v54.0.0)

Open the `App.js` file to start writing some code. You can preview the changes directly on your phone or tablet on **Expo go app** by scanning the **QR code** on Expo snack platform, or use the iOS or Android emulators. 

Snack is Open Source. You can find the code on the [GitHub repo](https://github.com/expo/snack).
